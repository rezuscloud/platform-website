# "Personal Cloud" as product identity

We call RezusCloud "the Personal Cloud" to parallel the Personal Computer revolution. The PC was personal not because one person used it, but because you owned it instead of renting mainframe time. The Personal Cloud is the same shift: you own the infrastructure, data, and computation instead of renting from a cloud provider. The term signals ownership and a paradigm shift, not scale or headcount.

Rejected alternatives: "private cloud" (implies network isolation, not ownership), "home cloud" (implies hobby scale), "self-hosted cloud" (describes the method, not the paradigm).
